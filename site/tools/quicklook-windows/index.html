QuickLook

QuickLook 为 Windows 提供类似 macOS 的空格键文件快速预览体验。

HAN'S VERDICT

为什么值得关注

QuickLook 已列入当前主力工具，适合在明确需求下作为日常工作流的一部分。

适合场景

希望在 Windows 文件管理器中快速预览图片、文档和媒体文件的用户。

替代方案

Microsoft PowerToys Peek

主要优点

  • 开源并提供官方仓库更新渠道
  • 用空格键快速预览多种文件

使用限制

  • 具体功能、系统要求和价格可能随版本调整
