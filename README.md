# CosmicHAN deployment

This public repository contains generated static deployment output for <https://han-cosmic.github.io/>.

The Astro source, Markdown content, maintenance documentation, and development history are maintained separately in a private repository. Files under `site/` are generated artifacts and should not be edited by hand.

This repository is a deployment target, not a source backup. Recover or change content in the private source repository, rebuild it, and replace the generated `site/` output.
